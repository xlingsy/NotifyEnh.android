# 更新日志 (Changelog)

## v26.06.10

- **新增：** 全屏响铃界面支持通过音量键（+ 或 -）快速停止响铃。
- **优化：** 解决重复通知取消任务未执行的问题。

## v26.06.06

- **新增：** 持续响铃功能。触发后可像闹钟一样响铃，支持全屏提醒界面和锁屏显示。
- **新增：** 铃声配置。支持为每个任务单独选择系统闹钟铃声。
- **新增：** 自动备份。每次任务编辑后会自动生成 JSON 备份，支持一键恢复。
- **优化：** 通知记录查重。过滤组摘要通知并增加内存级去重逻辑，解决记录重复问题。
- **修复：** 修复了部分系统下通知监听服务由于内存回收导致的中断问题。

## v26.05.29

- **基础功能：** 实现通知监听与本地数据库记录。
- **任务系统：** 支持基于包名、标题、内容的过滤规则。
- **自动化操作：** 支持自动取消通知和 TTS 语音播报。
- **配置管理：** 支持主题切换、记录保留时间设置、数据导出与导入。
